# Old Cache Data (moved 2025-06-30)

## Problem
The DDA cache (`_dda_cache/`) contained 291 JSON solve-cache files but only 3 could be
matched to known parameter combinations using the current `dda_sweep.py` hash-key logic.
The hash-key format (`v=v1_er=...`) matched only cube, er=10000 at 3 d-levels.
The remaining 288 files could not be resolved to (shape, er, d) parameters.

Possible causes:
- The sweep may have been run with a different `shape_grid()` or `er_values` than what
  is currently in `tools/dda_sweep.py`
- The `_CACHE_VERSION` string or floating-point format may have changed
- Cache files may have been generated by a different script entirely

## Contents
- `_dda_cache/` — original DDA sweep cache (291 files, ~June 28-29)
- `_mom2d_cache/` — original 2D MoM cache
- `_mom3d_cache/` — original 3D MoM cache
- `_dda_cache_tools/` — empty `tools/_dda_cache/` directory

## Resolution
All cache data moved here. Fresh recomputation starting from:
- 2D MoM (23 aspect ratios × 5 permittivities = 115 data points)
- Then 3D MoM, then DDA sweep
